html

The HTML element represents the root (top-level element) of an HTML document, so it is also referred to as the root element. All other elements must be descendants of this element.

Chlid: no

Attribute: no

Example: <!DOCTYPE html> <html> </html>

head

The head element contains machine-readable information (metadata) about the document, like its title, scripts, and style sheets.

Chlid: no

Attribute: no

Example: <!doctype html> <html> <head> <title>Document title</title> </head> </html>

body

The body element represents the content of an HTML document. There can be only one body element in a document

Chlid: no

Attribute: no

Example: <!doctype html> <html> <head> </head> <body> <p>This is a paragraph</p> </body> </html>

div

the div element is the generic container for flow content. It has no effect on the content or layout until styled in some way using CSS (e.g. styling is directly applied to it, or some kind of layout model like Flexbox is applied to its parent element).

Chlid: no

Attribute: no

Example: <!doctype html> <html> <head> </head> <body> <div> <p>Any kind of content here. Such as &lt;p&gt;, &lt;table&gt;. You name it!</p> </div> </body> </html>

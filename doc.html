HTML: HyperText Markup Language

The HTML element represents the root (top-level element) of an HTML document, so it is also referred to as the root element. All other elements must be descendants of this element.

Chlid: One head element, followed by one body element.

Attribute: No

Example:

<!DOCTYPE html> <html lang="en"> <head>...</head> <body>...</body> </html>

head : The Document Metadata (Header) element

The head element contains machine-readable information (metadata) about the document, like its title, scripts, and style sheets.

Chlid: One titile element

Attribute: No

Example:

<!doctype html> <html> <head> <title>Document title</title> </head> </html>

body : The Document Body element

The body element represents the content of an HTML document. There can be only one body element in a document

Chlid: all the contents of an HTML document, such as headings, paragraphs, images, hyperlinks, tables, lists, etc.

Attribute: No

Example:

<html> <head> <title>Title of the document</title> </head> <body> <h1>This is a heading</h1> <p>This is a paragraph.</p> </body> </html> </html>

h1 to h6 : The HTML Section Heading elements

The h1 to h6 HTML elements represent six levels of section headings. h1 is the highest section level and h6 is the lowest.

Chlid: No

Attribute: No

Example:

<h1>This is heading 1</h1> <h2>This is heading 2</h2> <h3>This is heading 3</h3> <h4>This is heading 4</h4> <h5>This is heading 5</h5> <h6>This is heading 6</h6>

p : The Paragraph element

The p tag defines a paragraph.

Chlid: No

Attribute: No

Example:

<p> My Bonnie lies over the ocean. My Bonnie lies over the sea. My Bonnie lies over the ocean. Oh, bring back my Bonnie to me. </p>

b : The Bring Attention To element

The b tag specifies bold text without any extra importance. According to the HTML5 specification, the b tag should be used as a LAST resort when no other tag is more appropriate.

Chlid: No

Attribute: No

Example:

<p>This is normal text - <b>and this is bold text</b>.</p>

strong : The Strong Importance element

The strong tag is used to define text with strong importance. The content inside is typically displayed in bold.

Chlid: No

Attribute: No

Example:

<strong>This text is important!</strong>

u : The Unarticulated Annotation (Underline) element

The u tag represents some text that is unarticulated and styled differently from normal text, such as misspelled words or proper names in Chinese text. The content inside is typically displayed with an underline.

Chlid: No

Attribute: No

Example:

<p>This is some <u>mispeled</u> text.</p>

i : The Idiomatic Text element

The i tag defines a part of text in an alternate voice or mood. The content inside is typically displayed in italic. The i tag is often used to indicate a technical term, a phrase from another language, a thought, a ship name, etc.

Chlid: No

Attribute: No

Example:

<p><i>Lorem ipsum</i> is the most popular filler text in history.</p> <p>The <i>RMS Titanic</i>, a luxury steamship, sank on April 15, 1912 after striking an iceberg.</p>

br : The Line Break element

The br tag inserts a single line break. The br tag is useful for writing addresses or poems. The br tag is an empty tag which means that it has no end tag.

Chlid: No

Attribute: No

Example:

<p> O’er all the hilltops<br> Is quiet now,<br> In all the treetops<br> Hearest thou<br> Hardly a breath;<br> The birds are asleep in the trees:<br> Wait, soon like these<br> Thou too shalt rest. </p>

hr : The Thematic Break (Horizontal Rule) element

The hr HTML element represents a thematic break between paragraph-level elements: for example, a change of scene in a story, or a shift of topic within a section.

Chlid: No

Attribute: No

Example:

<p>§1: The first rule of Fight Club is: You do not talk about Fight Club.</p> <hr> <p>§2: The second rule of Fight Club is: Always bring cupcakes.</p>

img : The Image Embed element

The img tag is used to embed an image in an HTML page. Images are not technically inserted into a web page; images are linked to web pages.

Chlid: No

Attribute: src (Specifies the path to the image), alt (Specifies an alternate text for the image, if the image for some reason cannot be displayed)

Example:

<img src="img_girl.jpg" alt="Girl in a jacket" width="500" height="600">

div : The Content Division element

The div tag defines a division or a section in an HTML document.

Chlid: No

Attribute: No

Example:

<div class="warning"> <img src="/media/examples/leopard.jpg" alt="An intimidating leopard."> <p>Beware of the leopard</p> </div>

table : The Table element

The table tag defines an HTML table. An HTML table consists of one table element and one or more , , and elements. The tr element defines a table row, the th element defines a table header, and the td element defines a table cell.

Chlid: tr ,th, td

Attribute: align, valign, width , height, boder , cellspacing

Example:

<!DOCTYPE html> <html> <head> </head> <body> <h1>The table element</h1> <table> <tr> <th>Month</th> <th>Savings</th> </tr> <tr> <td>January</td> <td>$100</td> </tr> <tr> <td>February</td> <td>$80</td> </tr> </table>
